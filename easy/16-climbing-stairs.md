爬楼梯问题：假设有 n 级台阶，每次可以爬 1 个或 2 个台阶，有多少中走法。

这个问题其实挺有意思的，一开始想了蛮久的，后面看了一句话，就觉得理解了

> 动态规划：最优解可以从其子问题的最优解来有效地构建

具体代码如下
```js
var climbStairs = function(n) {
    const dp = [];
    dp[1] = 1;
    dp[2] = 2;
    for(let i = 3; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
};
```
